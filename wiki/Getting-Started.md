# Getting Started

## Install

```bash
git clone https://github.com/dmoliveira/utils-scripts.git
cd utils-scripts
make help
make install-mac
```

Use `make install-debian` or `make install-unix` if needed.

## Verify

```bash
make verify
make doctor
```

## Enable hooks

```bash
make hooks-install
make pre-commit-install
```

## Learn workflows

- read `TERMINAL_PLAYBOOK.md`
- browse docs in `docs/`
- use cheatsheets in `docs/cheatsheets/`

# Top 10 New User Commands

Use these first when setting up your environment.

1. `make help` - discover supported install and verify targets.
2. `make install-mac` - install on macOS.
3. `make install-debian` - install on Debian/Ubuntu.
4. `make install-unix` - fallback installer for generic Unix systems.
5. `make verify` - run smoke checks after install.
6. `make doctor` - get strict checks with guided fixes.
7. `make hooks-install` - install repository git hooks.
8. `make pre-commit-install` - enable pre-commit hooks.
9. `make docs-browse` - browse docs quickly with preview.
10. `make rollback-dry-run` - preview rollback before applying changes.

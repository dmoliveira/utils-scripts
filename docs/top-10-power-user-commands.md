# Top 10 Power User Commands

Use these when you want speed, reliability, and repeatable delivery.

1. `tmux-delivery` - open a delivery-focused tmux workspace.
2. `tmux-incident` - open triage-focused tmux layout.
3. `watchexec -e py,sh,lua,md 'make verify'` - continuous validation loop.
4. `make doctor-full` - full local quality + strict verification gate.
5. `make wiki-build` - generate wiki mirror payload locally.
6. `make wiki-build-check` - verify sidebar/index mirror rules.
7. `trivy fs .` - quick repository security scan.
8. `hyperfine --warmup 1 '<old>' '<new>'` - benchmark command alternatives.
9. `pre-commit run --all-files` - run all configured local gates.
10. `gh pr create` - open PRs with clear summary and validation evidence.

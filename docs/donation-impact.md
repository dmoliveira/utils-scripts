# Donation Impact

Thank you for considering support. This page explains where support goes and why it matters.

## What your support unlocks

- faster fixes when package managers or platform defaults change
- improved verification scripts and clearer doctor hints
- better onboarding docs for new users
- deeper cheatsheets and playbooks for power users

## Typical maintenance work funded by donations

- keeping installers stable across macOS and Debian/Ubuntu updates
- adding reliable fallbacks when tools are missing or renamed
- reducing CI false negatives in docs and release pipelines
- expanding practical examples for incident and delivery workflows

## If you cannot donate

You still help a lot by opening high-quality issues, sharing the project, and contributing reproducible fixes.
